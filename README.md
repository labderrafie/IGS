# IGS - Site Web Professionnel

Site web pour IGS - Intégrale Géophysique Système, entreprise spécialisée dans la recherche d'eau et l'analyse des sols.

## 🌐 Fonctionnalités

- **Design responsive** : Mobile, tablette et desktop
- **Formulaire de contact** : Intégration Netlify Forms
- **Navigation intuitive** : Menu hamburger sur mobile
- **Design moderne** : Animations CSS et interface épurée

## 📱 Pages

- **Accueil** : Présentation des services et CTA
- **À propos** : Histoire, mission et expertise de l'entreprise
- **Services** : Recherche d'eau, puits/forages, analyse des sols
- **Contact** : Formulaire de contact et informations

## 🛠️ Technologies

- HTML5 / CSS3 / JavaScript
- Netlify Forms (formulaire de contact)
- Support RTL pour l'arabe

## 🚀 Déploiement

Déployé sur Netlify avec domaine personnalisé : [Lien vers le site](https://www.igs-prospection.com/)
