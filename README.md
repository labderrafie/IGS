# Site Web IGS - Intégrale Géophysique Système

## Description
Site web professionnel pour IGS, entreprise spécialisée dans la recherche et l'exploration des ressources en eau, la gestion de puits et forages, et l'analyse des sols agricoles.

## Structure du site

### Pages principales
- **index.html** - Page d'accueil avec présentation des services
- **about.html** - À propos de l'entreprise, équipe, expertise
- **services.html** - Détail des services proposés
- **equipment.html** - Présentation de l'équipement technique
- **contact.html** - Formulaire de contact et informations

### Dossiers
- **css/** - Feuilles de style (style.css)
- **js/** - Scripts JavaScript (main.js)
- **images/** - Images et photos (à ajouter)

## Fonctionnalités

### Design
- Design responsive (mobile, tablette, desktop)
- Navigation mobile avec menu hamburger
- Animations CSS au scroll
- Couleurs professionnelles (bleu #2c5aa0, orange #ff6b35)

### Pages
- **Accueil** : Hero section, services principaux, équipements, CTA
- **À propos** : Histoire, mission, équipe, certifications
- **Services** : Recherche d'eau, puits/forages, analyse des sols
- **Équipements** : Présentation technique détaillée
- **Contact** : Formulaire, informations, FAQ

## Personnalisation

### Images à ajouter
Placez vos images dans le dossier `images/` avec ces noms :
- `hero-equipment.jpg` - Image principale de l'équipement
- `about-team.jpg` - Équipe en action
- `service-*.jpg` - Images pour chaque service
- `equipment-*.jpg` - Photos de l'équipement
- `team-member-*.jpg` - Photos de l'équipe

### Informations à personnaliser
1. **Coordonnées** dans `contact.html` :
   - Téléphone
   - Email
   - Adresse

2. **Équipe** dans `about.html` :
   - Noms et rôles
   - Photos

3. **Contenu** :
   - Modifiez les textes selon vos besoins
   - Ajoutez vos certifications
   - Personnalisez les témoignages

## Déploiement

### Option 1 : Hébergement simple
1. Uploadez tous les fichiers sur votre serveur web
2. Le site fonctionne immédiatement

### Option 2 : GitHub Pages (gratuit)
1. Créez un repository GitHub
2. Uploadez les fichiers
3. Activez GitHub Pages dans les paramètres

### Option 3 : Netlify (gratuit)
1. Connectez votre repository GitHub
2. Netlify déploie automatiquement

## Maintenance

### Ajout d'images
1. Ajoutez vos images dans le dossier `images/`
2. Mettez à jour les références dans le HTML si nécessaire

### Modification du contenu
1. Éditez les fichiers HTML directement
2. Les changements sont immédiats

### Mise à jour du design
1. Modifiez `css/style.css`
2. Testez sur différents appareils

## Support

Le site est conçu pour être simple à maintenir. Tous les fichiers sont commentés et la structure est claire.

Pour des modifications avancées, consultez la documentation HTML/CSS ou contactez un développeur web.
